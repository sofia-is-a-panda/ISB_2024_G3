Git init --> Crear un repositorio personal con Git en mi PC.
Git add --> Rastrear un archivo con git.
Git commit --> Añadir el archivo a una "Nube personal".
Git push --> Añadir el archivo al repositorio en Github. Se tiene que especificar la rama del repositorio a la cual
se quiere hacer cambios.
Git pull --> Descargar los archivos de un repositorio en Github hacia mi repositorio personal, tomando en cuenta
que el historial de versiones de tu repositorio personal coincida con el historial de versiones del repositorio en Github.
Git clone --> Clonar un repositorio de Github a mi repositorio personal. Esto se usa cuando vas a "conectar" 
un repositorio de Github a tu repositorio personal.
